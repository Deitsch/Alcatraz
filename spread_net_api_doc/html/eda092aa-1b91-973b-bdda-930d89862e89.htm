Join Method (c, groupName)
Namespaces ► spread ► SpreadGroup ► Join(SpreadConnection, String)
Joins the group groupName on the connection c. Between when the group has been joined with this method and when it is left with Leave()()(), all messages sent to the group will be received by the connection.
