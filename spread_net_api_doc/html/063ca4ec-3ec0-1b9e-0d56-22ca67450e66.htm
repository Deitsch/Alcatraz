if(thisID.Equals(thatID) == true)

SpreadConnection
A SpreadConnection object is used to establish a connection to a spread daemon. To connect to a spread daemon, first create a new SpreadConnection object, and then call Connect(String, Int32, String, Boolean, Boolean):
CopyC#
SpreadConnection connection = new SpreadConnection();
connection.Connect(null, 0, "name", false, false);
 The only initialization before Connect(String, Int32, String, Boolean, Boolean) are to add and remove SpreadConnection..::.MessageHandlers, see OnRegularMessage and OnMembershipMessage. If any other methods are called, a SpreadException is thrown, except for PrivateGroup, which returns null.
To disconnect from the daemon, call Disconnect()()():
CopyC#
connection.Disconnect();
 To send a message on this connection, call Multicast(SpreadMessage):
CopyC#
connection.Multicast(message);
 To receive a message sent to this connection, call Receive()()():
CopyC#
SpreadMessage message = connection.Receive();

SpreadGroup
A SpreadGroup object represents a group on a spread daemon. The group is either a group that has been joined or a private group. To join a group, first create a SpreadGroup object, then join it with Join(SpreadConnection, String):
CopyC#
SpreadGroup group = new SpreadGroup();
group.join(connection, "users");
 To leave the group, use Leave()()():
CopyC#
group.leave();

SpreadMessage
A SpreadMessage object represents either an incoming or outgoing spread message. An outgoing message is one being sent with Multicast(SpreadMessage). An incoming message is one received with Receive()()(). To send a message on a spread connection, first create a message object:
CopyC#
SpreadMessage message = new SpreadMessage();
 Then set the data: Data
CopyC#
message.Data = data;
 Select which group(s) to send the message to with AddGroup(SpreadGroup):
CopyC#
message.AddGroup(group);
 When the message is read to be sent, send it with Multicast(SpreadMessage):
CopyC#
connection.Multicast(message);
