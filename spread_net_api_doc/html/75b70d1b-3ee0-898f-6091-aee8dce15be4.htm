SpreadMessage Class
Namespaces ► spread ► SpreadMessage
A SpreadMessage object represents either an incoming or outgoing spread message. An outgoing message is one being sent with Multicast(SpreadMessage). An incoming message is one received with Receive()()(). To send a message on a spread connection, first create a message object:
CopyC#
SpreadMessage message = new SpreadMessage();
 Then set the data: Data
CopyC#
message.Data = data;
 Select which group(s) to send the message to with AddGroup(SpreadGroup):
CopyC#
message.AddGroup(group);
 When the message is read to be sent, send it with Multicast(SpreadMessage):
CopyC#
connection.Multicast(message);
